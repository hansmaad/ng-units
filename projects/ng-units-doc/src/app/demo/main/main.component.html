Bootstrap input group

This input group is using ngQuantity on the input to automatically convert values to the currently selected unit and ngUnitSelect to easily fill a HTML select with available units. 

<input [(ngModel)]="value" ngQuantity="Temperature" >

The next input uses ngUnit to specify a fixed unit. No matter which unit is currently selected for the Temperature quantity, it will always use 째C for the conversion. 

<input [(ngModel)]="value" ngQuantity="Temperature" ngUnit="째C" >
